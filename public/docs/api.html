API v1

Tutte le API sono sullo stesso dominio. Autenticazione JWT via header Authorization: Bearer <token>.

Auth

  • POST /api/v1/auth/login
  • GET /api/v1/auth/me

Agenzia & Collaboratori

  • POST /api/v1/agency/register
  • POST /api/v1/collaborators
  • GET /api/v1/collaborators
  • PUT /api/v1/collaborators/{id}
  • DELETE /api/v1/collaborators/{id}
  • GET /api/v1/permissions

Immobili

  • POST /api/v1/immobili
  • GET /api/v1/immobili
  • PUT /api/v1/immobili/{id}
  • DELETE /api/v1/immobili/{id}

Clienti & Lead

  • POST /api/v1/lead

Appuntamenti
